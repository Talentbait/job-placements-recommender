Before running the app, update and install current versions:
    pip install -r requirements.txt
    
Also download and move the file case_sensitive to the folder recommender_system/models/
    https://drive.google.com/drive/folders/1jZh79hf9p92_fsAh3J-LGO-hX6qpzAI7

To run app locally, use:
    streamlit run text_classification_and_collaborative_recsys.py
